防火墙操作命令

解释 命令
开启防火墙 systemctl start firewalld
关闭防火墙 systemctl stop firewalld
重启防火墙 systemctl restart firewalld
重启防火墙 firewall-cmd –reload
开启端口 firewall-cmd –zone=public –add-port=10002/tcp –permanent
检查修改是否生效 firewall-cmd –zone= public –query-port=10002/tcp
移除指定端口 firewall-cmd –permanent –remove-port=123/tcp
查看已开放端口 firewall-cmd –zone=public –list-ports
查看防火墙状态 systemctl status firewalld
看指定端口号的进程情况 netstat -tunlp l grep 端口号
查看端口的连接客户端IP netstat -anp l grep 端口号
查看端口的连接客户端IP netstat -anp
查看某一端口的占用情况 lsof -i:端口号

log日志查看

解释 命令
用于显示文件的末尾内容 tail -n 100 filename
用于显示文件的开头内容 head -n 100 filename
日志查询 tail -fn 1000 test.log
循环读取日志 tail -f error.log

docker 常用命令

解释 命令
查看本地镜像列表 docker images
列出所有在运行的容器信息 docker ps
启动一个容器 docker start 容器Id
停止一个运行中的容器 docker stop 容器Id
重启容器 docker restart 容器Id
杀掉一个运行中的容器 docker kill tomcat7
删除一个 docker rm -f db01
强制删除本地镜像 docker rmi -f 容器id
